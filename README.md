# AI-Powered-Lead-Data-Extraction-Agent
This n8n automation agent fetches live data from Google APIs, processes and transforms it using custom JavaScript logic, and automatically creates a structured Google Sheet — ready for BI dashboards or CRM ingestion.

🤖 I built an AI agent that turns Google API data into a structured report — fully automated, zero manual work.

Here's exactly how it works 👇

━━━━━━━━━━━━━━━━━━━━━━━━

⚙️ THE WORKFLOW (7 nodes, end-to-end):

1️⃣ Trigger — Manual or scheduled execution
2️⃣ Edit Fields — Sets search parameters (query, location, filters)
3️⃣ HTTP Request — Pulls live data from Google API
4️⃣ JavaScript — Cleans, filters & transforms raw JSON into structured records
5️⃣ Create Spreadsheet — Auto-generates a fresh Google Sheet per run
6️⃣ Append Row — Writes primary data fields
7️⃣ Append Row — Writes secondary/metadata fields
